Commands.

TotoBot only uses slash (/) commands. There is no need to get confused with prefixes!
Fields with [] are required, {} are optionals.

Command Description Usage
/summoner Get various information about a LoL player. /summoner [player name]
/play Play music. /play[song name]
/skip Skip the current song. /skip
/queue See the server music queue. /queue
/jump Jump to an specific song in the queue. /jump [Number]
/loop Loop the queue. /loop [Song,Queue,Off]
/remove Remove a song from the queue. /remove [Number]
/shuffle Shuffle the queue. /shuffle
/clear Clear the queue. /clear
/ping Pong. /ping
