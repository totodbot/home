CakeBot is a Discord bot with Mojang api utilities, it also has more commands from different categories.

CakeBot only uses slash (/) commands. There is no need to get confused with prefixes!
Fields with [] are required, {} are optionals.

Command Description Usage
/player Get various information about a Minecraft player. /player [player name]
/head Get the head of a minecraft player skin. /head [player name] {True/False}
/skin Get the skin of a minecraft player. /skin [player name] {True/False}
/server See the status of a minecraft server. /server [server ip]
/advancement Generate an "advancement get!" image. /advancement [text]
/mob Sends a random minecraft mob image. /mob
